##Shell is kernel's Shell

1. 分屏 使用分屏可以在一个终端中查看、操作一个较长的文件或者查看多个文件


| 命令              | 解释      |
| ---------------- | ---------------- |
| :sp [filename]   | 分屏，打开一个新窗口，没有文件时打开本文件 |
| [ctrl]+w+↑       | 切换到上面的屏                       |
| [ctrl]+w+↓       | 切换到下面的屏                       |
| [ctrl]+w+q       | 离开                               |
